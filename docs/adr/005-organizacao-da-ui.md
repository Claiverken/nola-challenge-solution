# ADR-005 — Organização da Interface e Reutilização
**Data:** 03/11/2025  
**Status:** Implementado  

## 1. Contexto  
A primeira versão do painel estava poluída e pouco intuitiva.

## 2. Decisão  
Organizar a interface com **seções (Tabs)** e **componentes reutilizáveis** (`InfoList.vue`, `ChartCard.vue`, etc.).

## 3. Consequências  
- ✅ Melhor UX e navegação.  
- ✅ Código modular e limpo.  
- ❌ Requer padronização entre componentes.
