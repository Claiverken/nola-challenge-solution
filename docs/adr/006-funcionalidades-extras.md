# ADR-006 — Funcionalidades Extras e Valor de Produto
**Data:** 03/11/2025  
**Status:** Implementado  

## 1. Contexto  
O projeto deveria oferecer insights além do básico, agregando valor real para o usuário.

## 2. Decisão  
Foram implementados endpoints e widgets adicionais:  
- **Top Itens Adicionais**  
- **Performance de Entrega (por região)**  
- **Clientes em Risco (RFM)**  
- **Top Clientes VIP**

## 3. Justificativas  
- Atender necessidades identificadas nos requisitos.  
- Demonstrar visão de produto.

## 4. Consequências  
- ✅ Entrega de valor superior.  
- ✅ Maior utilidade para o usuário.  
- ❌ Aumento da complexidade de manutenção.
